How to update this tracker
  1. Open index.html in any text editor and scroll to the line that says const PLAN = [.
  2. Find the week you ran, then the day (Mon, Wed, Fri, or Sat).
  3. Add an actual entry like this:
    {d:'Fri', plan:5, actual:{dist:5.02, time:'53:40', pace:'10:41', hr:152, loc:'Forest Acres', temp:60}}
  4. Save the file. The checkmark, miles, and progress bar update automatically.
  5. To publish: in your GitHub repo choose Add file → Upload files, drag in the new index.html, and commit. The live site refreshes in a couple of minutes.
